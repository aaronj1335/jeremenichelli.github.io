These last weeks weren't the best, both personal and work life. Details aren't important.

I've always wanted to have a blog but everytime a lot of questions came to my mind like will you have the time to write posts? or do you really have something to write about? and my all time favorite are you sure?.

Well, the thing is that you're reading this and that means that a few months ago I asked myself the same questions and suddenly the answer were yes, yes and my all time favorite yes

You have to try

I had some ideas about what the next post could be, I still have them noted down just in case. Sadly there were always a problem or an event I had to (and wanted to) go that was getting in my way. Honestly, some days I wanted but I wasn't in the right mood to do it.

But I tried. I tried with one idea, but couldn't wrap it. For other post I had to do a lot of scripts and testing and didn't have the time. One saturday morning I was going to write down about some approach I used a long time ago, an interesting one actually, but I needed time to make some investigation because performance matters now. I mean, performance always mattered but I wasn't the developer I am today in that time. Not that I'm great now, but, well I'm sure you got my point.

I needed to investigate this and that's actually fun, but I didn't have the time to do it in that morning. I started, and since the thing I wanted to test involved loops and I wanted to measure time, just like in my life. Well I wanted to test execution time in this case.

The script

So, I said to myself, take it cool. Enjoy it! And I did it actually. I had to write something like a chronometer, something that started counting and at any given moment log how much time has passed. And I did it actually.

The result of all of this is Chronos, a script that allows you to measure how many time has passed since you initialized it. You can create all the Chronos instances you wanted using the word new and you can put checkpoints wherever you want and as much as you need. You can label them to have an idea of what is in consideration regarding that particular checkpoint.

I'm pretty sure I'm going to add a lot of new things so stay tuned. Of course I'm open to suggestions so if you have some just throw them on the repository.

Wrap up

I enjoy having a blog a lot, even if I can't write as much as I would like to. If you're reading this, thanks because is great to have you around and means a lot. Really. And if you think you need more time to do things you like just try, just start doing them. Measure your time. And if you need to measure the time in your scripts you have Chronos.

Hope to see you around soon and happy coding!
