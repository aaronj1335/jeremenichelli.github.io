Experience

After some time working on small sites for companies that were settled on my home town while studying, I decided to make some changes and leave University because I wasn't really learning and enjoying it as I did before.

The next step was to become a self-taught web developer specialized in front end technologies.

That same year, I was hired by Globant and had the chance to work with brands like Microsoft, StateFarm and the New York Stock Exchange between others on beautiful and scalable responsive web projects.

Later, I joined OrderGroove to build awesome scripts that power the subscription revolution.
