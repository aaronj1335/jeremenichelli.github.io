Story and experience

In the past, I spent some time working on small sites for companies that were settled on my home town while studying, but decided to make some changes, quit my job and leave university because I wasn't really learning and enjoying it as I did before.

The next step was to become a self-taught front end developer.

That same year, I was hired by Globant and had the chance to work with big brands and companies like Microsoft, StateFarm and the New York Stock Exchange between others on beautiful and scalable responsive web projects.

Later, I joined OrderGroove to build awesome scripts that power the subscription revolution with their great team from New York.

About this site

This site was built using Jekyll and is hosted by GitHub Pages. If you're more intereseted in creating a site using Jekyll, I wrote some posts showing how to set up, create layouts and generate posts with it.

Styles and Design

This project uses LESS as a preprocessor for styles. The grid system was built using this little helper and the colors come from the Material Design style guide by Google.

Fonts

The font that rules the whole site is Roboto Condensed by Christian Robertson, interface designer at Google and the one from the logo is Southpaw crafted by Tyler Finck, whose work you should definitely check out by the way.

Writing

All the posts in my blog are written using Markdown.

Browser support

This site should run perfectly in the latest versions of Chrome, Firefox, Safari, Safari mobile, Chrome for Android, Internet Explorer 10 and 11 and Microsoft Edge.

If you see something that is not working good or that could be improved feel free to report it.
